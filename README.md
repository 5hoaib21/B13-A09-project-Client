## 🎯 Purpose

The main goal of **Study Nook** is to help students find and book peaceful study environments easily. It provides a smooth and reliable booking experience with real-time conflict prevention.

---

## ✨ Key Features

- 📱 Fully Responsive Design (Mobile / Tablet / Desktop)
- 🔐 Secure Authentication (Email & Google Login)
- 🏠 Room Management (Add / Update / Delete)
- 📅 Smart Booking System with Date & Time Selection
- 🚫 Booking Conflict Prevention (No double booking)
- 👤 User Profile Management
- 🔔 Toast Notifications for better UX
- ⚡ Clean and Modern UI Design
